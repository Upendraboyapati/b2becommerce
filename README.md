**B2B E-Commerce Platform**
A scalable Business-to-Business (B2B) e-commerce application designed for manufacturers, wholesalers, and distributors to manage inventory, orders, and role-based access securely.
Built with a modern full-stack architecture using React + Spring Boot + JWT + RBAC + MySQL. Tech Stack
Frontend
React (Vite)
Tailwind CSS
Axios
React Router
Backend
Spring Boot
Spring Security
JWT Authentication
Role-Based Access Control (RBAC)
REST APIs
JPA / Hibernate
Database
MySQL
**Core Features**
Authentication & Authorization
JWT-based secure login/signup
Role-Based Access Control (Admin, Vendor, Buyer)
Protected API routes
Token validation & expiration handling
Admin Module
Manage users (assign roles)
Product category management
Order oversight
Dashboard analytics (future enhancement ready)
**Vendor Module**
Add / Update / Delete products
Inventory tracking
Order processing
Stock updates
**Buyer Module**
Browse products
Add to cart
Bulk ordering
Order history tracking
